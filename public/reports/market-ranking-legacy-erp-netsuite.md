## Corridor 1: Epicor → NetSuite

### Epicor On-Premise Sunset Timeline (Critical Market Catalyst)

| Product | Final On-Prem Release | Active Support Ends | Sustaining Support |
|---------|----------------------|--------------------|--------------------|
| Epicor Kinetic | 2028.1 (Jan 2028) | Dec 31, 2029 | Indefinite (limited) |
| Prophet 21 | 2028.1 (May 2028) | Jun 2029 | Limited |
| BisTrack | 2027.1 (Apr 2027) | Apr 2028 | Limited |

**Implication:** 21,000+ Epicor customers face forced cloud migration or system replacement by 2029. This is the single largest ERP migration catalyst in mid-market manufacturing.

### NetSuite vs Epicor Comparison (BrokenRubik data)

| Category | NetSuite | Epicor Kinetic |
|----------|----------|----------------|
| Heritage | Cloud financials, expanded to manufacturing | Manufacturing ERP since 1972 |
| Deployment | Cloud-only SaaS | Cloud, hybrid, or on-premise |
| Target Market | $10M-$500M, multi-industry | $10M-$500M, manufacturing/distribution |
| Base Pricing | ~$999/mo + $99-199/user/mo | Custom; ~$175-250/user/mo est. |
| Manufacturing Depth | Moderate (Advanced Mfg module) | Deep (purpose-built MRP, APS, MES) |
| Financial Management | Strong (multi-entity, ASC 606, global tax) | Adequate (solid core, less global) |
| Built-in CRM | Yes | Basic |
| Built-in eCommerce | SuiteCommerce | Epicor Commerce |
| Cloud Architecture | True multi-tenant SaaS | Azure-hosted, hybrid options |
| Customization | SuiteScript (JavaScript) | Epicor Functions (C#/.NET) |
| Implementation Time | 3-6 months | 4-9 months |

### First-Year Cost Estimates (30 users)
- **NetSuite:** $80,000-$200,000
- **Epicor:** $100,000-$300,000

### Pro-Flex Case Study (AlphaBold)
- **Company:** Pro-Flex (plumbing products manufacturer)
- **From:** Epicor (on-premise)
- **To:** NetSuite cloud ERP
- **Reasons:** Legacy limitations, no real-time info, needed automation/transparency
- **Results:** Streamlined operations, automated reporting, full transparency across financials/inventory/production, improved forecasting, reduced manual effort

### When Epicor Wins Over NetSuite
- Job shops and make-to-order manufacturers
- Complex BOMs and routings
- Regulated industries requiring formal quality systems (ISO, FDA)
- Companies where manufacturing complexity is the PRIMARY business challenge

### When NetSuite Wins Over Epicor
- Wholesale distribution + ecommerce + services (multi-channel)
- Multi-subsidiary financial management
- Manufacturing as ONE PART of a broader operation
- Companies prioritizing speed of implementation

## Corridor 2: Infor → NetSuite

### Infor Product Landscape

| Product | Status | Target Market |
|---------|--------|---------------|
| Infor XA (formerly MAPICS) | Legacy AS/400 — acute skills shortage | Discrete manufacturing |
| Infor SyteLine / CloudSuite Industrial | Active, Leap migration available | Manufacturing |
| Infor CloudSuite | Active cloud offering | Multi-industry |
| Infor M3/LN | Enterprise-grade, end-of-life discussions | Large manufacturers |
| Infor SunSystems/FMS | Legacy financials | Services, NGOs |
| Infor VISUAL | Legacy — seeking replacements | Manufacturing |

### Why Manufacturers Leave Infor XA (MSDynamicsWorld)
1. Outdated green-screen UI (AS/400 architecture)
2. Limited scalability
3. Difficulty integrating modern tools (CRM, eCommerce, WMS)
4. High cost of customizations
5. Limited cloud capabilities
6. Skills shortage — RPG and green-screen developers retiring
7. Complex customization layers accumulated over decades
8. Higher long-term maintenance costs

### Infor XA Alternatives Ranked (MSDynamicsWorld)
1. **D365 Business Central** — Best overall (deep manufacturing, lower cost, Microsoft ecosystem)
2. **D365 Finance & Operations** — For large, multi-entity operations
3. **Epicor Kinetic** — Strong production features
4. **Oracle NetSuite** — Fully cloud-native, but manufacturing needs add-ons
5. **SAP Business One** — Solid MRP, good financial control
6. **Acumatica** — Licensing flexibility, modern web interface
7. **Infor CloudSuite Industrial** — Infor-to-Infor upgrade path

**Note:** NetSuite ranked #4 for Infor XA replacement — manufacturing add-on requirement is a weakness. Source should address this head-on in content.

### Infor vs NetSuite Comparison (Folio3)

| Area | Infor | NetSuite |
|------|-------|----------|
| Deployment | Industry-specific, hybrid/cloud | Cloud-first SaaS |
| Implementation | 9-18 months | 3-6 months |
| Pricing | Custom, varies by module | ~$999/mo + $99/user/mo |
| Best For | Deep vertical manufacturing | Integrated suite (finance + inventory + CRM + ecommerce) |
| Customers | 68,000+ | 37,000+ |
| Manufacturing Depth | 35% more native manufacturing functionality | Moderate, needs Advanced Mfg module |

### Infor Leap Program
- Fixed-fee, fixed-timeline cloud migration from legacy Infor to Infor CloudSuite
- Directly competes with "switch to NetSuite" narrative
- Source content should acknowledge Infor Leap but highlight scenarios where NetSuite is superior (multi-channel, services component, ecommerce)

## Corridor 3: Legacy ERP → NetSuite (General)

### Hidden Costs of Legacy ERP (Forbes / Moor Insights)
1. **Operational Delays** — Limited visibility, slower decisions, manual scheduling
2. **Compliance Issues** — Difficulty locating audit records
3. **Technical Debt** — Years of custom fixes complicate upgrades
4. **Workforce Management** — Longer onboarding for new employees
5. **Supply Chain Inefficiencies** — Reactive vs proactive, last-minute freight costs

### 5 Risks of Staying on Legacy ERP (Cloud9ERP)
1. **Rising Costs** — Hardware update cycles, CapEx treadmill
2. **Lost Agility** — Inflexible, slow to customize
3. **Support Drying Up** — End-of-life, employee turnover/retirement
4. **Integration Challenges** — Lack of standards-based APIs
5. **Underutilized Data** — No built-in analytics or reporting

### Legacy ERP Drawbacks (CohnReznick)
- Limited support in sunsetting phase
- Outdated infrastructure requiring costly hardware
- Data silos from poor integration
- No AI-enabled features
- Struggles with hybrid/decentralized work

### NetSuite Benefits for Legacy Migrants
- Customizable modular platform for growth
- Resource optimization with automation
- AI-driven productivity (generative AI features)
- Integrated ecosystem for all applications
- Future-ready capabilities

### ERP Vendor Lock-In Risks
- Proprietary data formats
- High switching costs designed into contracts
- Annual maintenance fee treadmill
- 3-year contract lock-ins becoming standard
- Panorama Consulting: "What does it cost to leave?" is the key question
