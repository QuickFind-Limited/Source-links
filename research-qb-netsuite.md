## PROTELO DEEP ANALYSIS (Why 25/25 despite minimal Google presence)

### Root Cause: Entity-Intent Density
- 380 sitemap URLs, 203 with "netsuite" in the URL
- Pages use direct buyer language: "implementation partner", "pricing", "migration"
- PR Newswire item from 2012 about joining NetSuite Solution Provider Program (durable training data)
- Ecwid blog "inventory management software" article has 29 localized URL variants referencing Protelo+NetSuite
- Reddit r/Netsuite partner threads mention Protelo positively

### Key Finding: Citations ≠ Recommendations
- **Citations** reward being a reusable evidence source across many answers (Anchor wins: 191)
- **Mentions** reward being strongly associated with "best partner for X" in model memory (Protelo wins initial)
- These are DIFFERENT SYSTEMS requiring separate optimization strategies

### Why Anchor Has 191 Citations But Fewer Mentions (20/25)
- Anchor is a broad content authority (hence high citations)
- But recommendation prompts diversify outputs across firms
- Protelo is a narrower, stronger entity-intent match → named more consistently

### Replication Strategy for Source
1. 8 core pages with exact prompt-aligned titles
2. 1 canonical cost page with explicit ranges + FAQ schema
3. 6 named migration case studies (industry, timeline, budget, outcomes)
4. 1 "Partner selection framework" page
5. Each core page in 3 formats: HTML + PDF + video transcript
6. Third-party reinforcement: G2/Clutch, ERP directories, Reddit presence
7. Track mention rate AND citation share as separate weekly KPIs
8. Optimize for mention rate first, then expand citation footprint
